SF Restaurants

This is a project in collaboration with Sandeep Vanga and Weiwei(Vivian) Zhang. Our application uses Shiny to visualize the operating hours of a random sample of restaurants in San Francisco for each day of the week. The interactive map gives an idea for which SF neighborhoods are awake and alive at a certain time. The GitHub Homepage contains the source codes for scraping Yelp pages to create a database of the opening-closing times of the SF restaurants, as well as the code for the Shiny application.
