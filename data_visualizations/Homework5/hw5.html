Data preprocessing


No data processing outside of javascript code is done. The data is downloaded from http://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/4.5_month.csv
.

Functionalities


Basic functionality
The basic code in http://bl.ocks.org/sjengle/eaba52747c044273d900 is properly modified to depict the earthquakes in the world.

Moderate functionality
The circled colors are modified by depth using threshold color sclae.

Functionality Discussion

Red bubbles show deep earthquakes and blue bubbles show shallow ones. Hovering on a bubble shows detailed information for the particular earthquake at the bottom of the map. I tried to distinguish overlapping bubbles as much as possible by adjusting the opacity and by sorting the bubbles by size before plotting. The radius of a bubble are proportional to the earthquake magnitude.
